常见问题

支持哪些相机 RAW 格式？

支持几乎所有主流相机品牌的 RAW 格式，包括 Nikon (NEF), Canon (CR2/CR3), Sony (ARW), Fujifilm (RAF), DNG 等。

彗星模式是什么？

彗星模式会创建渐变尾迹效果，让星轨看起来像彗星划过天空。可以调节尾巴长度（短/中/长）来获得不同的艺术效果。

生成的视频是什么格式？

生成 MP4 格式的视频，分辨率为 3840×2160 (4K)，帧率 25 FPS。包括星轨延时（展示星轨形成过程）和银河延时（展示天空运动）两种。

Windows 或 Linux 版本？

目前仅支持 macOS。代码是开源的，欢迎社区贡献其他平台的移植版本。
