## 图片要求
- **格式**: PNG 或 JPG
- **推荐尺寸**:
  - 教程截图: 1200×800 左右
  - 安装截图: 保持原始分辨率
  - 示例照片: 1920×1080 或更高
- **优化**: 建议压缩图片以加快网页加载速度
