彗星星轨 (SuperStarTrail) v0.5.0
一键生成星轨照片与延时视频

==================================================

安装说明：

1. 将 SuperStarTrail.app 拖拽到 Applications 文件夹
2. 首次运行：右键点击应用 → 选择"打开"
3. 如遇到安全提示，前往"系统偏好设置" → "安全性与隐私"允许运行

==================================================

快速开始：

1. 点击「选择目录」选择包含 RAW 文件的文件夹
2. 选择堆栈模式（Lighten 星轨 / Comet 彗星效果）
3. 配置参数（白平衡、延时视频等）
4. 点击「✨ 开始合成」，等待处理完成
5. 完成后自动打开输出目录查看结果

==================================================

主要功能：

✨ 原生 RAW 支持 - 支持所有主流相机格式
⭐ 多种堆栈模式 - Lighten、Comet、Average、Darken
🎬 延时视频 - 星轨延时 + 银河延时，4K 分辨率
🎨 实时预览 - 合成过程中实时查看结果
🔧 专业调控 - 白平衡调整、彗星尾巴长度

==================================================

系统要求：

- macOS 10.13 或更高版本
- 推荐 8GB 内存以上
