# UI 更新说明

## 名称调整

为了更准确地描述功能，我们对延时视频选项进行了重命名：

### 旧名称 → 新名称

- ❌ ~~延时视频~~ → ✅ **星迹延时**
- ❌ ~~普通延时视频~~ → ✅ **传统延时**

## 布局优化

将三个选项从两行改为一行显示，界面更加紧凑：

```
旧布局:
┌─────────────────────────────────┐
│ ✓ 间隔填充                       │
├─────────────────────────────────┤
│ ☐ 延时视频                       │
│ ☐ 普通延时视频                   │
└─────────────────────────────────┘

新布局:
┌─────────────────────────────────────────────────┐
│ ✓ 间隔填充   ☐ 星迹延时   ☐ 传统延时            │
└─────────────────────────────────────────────────┘
```

## 功能说明

### 1️⃣ 间隔填充
- **作用**: 填补星点之间的间隔，使星轨更连续
- **默认**: ✅ 启用

### 2️⃣ 星迹延时
- **作用**: 将星轨形成过程制作为延时视频
- **内容**: 展示从第1张到第N张的星轨变长过程
- **适合**: 展示星轨堆栈的过程
- **默认**: ☐ 禁用

### 3️⃣ 传统延时
- **作用**: 直接将原始照片合成为延时视频
- **内容**: 按时间顺序播放原始照片
- **适合**: 展示天空运动、云层变化等
- **默认**: ☐ 禁用

## 输出说明

用户可以同时获得：

| 选项 | 输出文件 | 说明 |
|------|---------|------|
| （默认） | `IMG_0001-0100_Lighten_CameraWB.tif` | 星轨堆栈图片 |
| 星迹延时 | `Timelapse_IMG_0001-0100_Lighten_25FPS.mp4` | 星轨形成过程 |
| 传统延时 | `SimpleTimelapse_IMG_0001-0100_CameraWB_25FPS.mp4` | 原始照片延时 |
